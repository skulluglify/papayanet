# PpNet (PapayaNet)

PpNet (PapayaNet) is a web framework for API server, built-in from library Echo, Gorm, Etc. Make it easy use for production, development, and testing code ...

![PapayaNet Showcase](docs/imgs/papayanet_showcase.png)

## Features

- Swag, Swagger
- Console, Koala
- Vector, Panda

## Built With

- Echo, Context
- Gorm, Postgres
